


- `Ctrl+S` / `Cmd+S`: Save file
- `Ctrl+E` / `Cmd+E`: Export as PDF
- `Ctrl+Shift+E` / `Cmd+Shift+E`: Export as HTML
